========================================================================
 《蠟像館之謎》Waxworks 繁體中文版 — 如何開始遊玩
========================================================================

本套件是「引擎包」：只含 patched ScummVM 引擎與繁體中文字型，
**不含遊戲本體**（尊重原作版權）。你需要自備一份合法擁有的
Waxworks (Floppy/DOS) 原始檔，放進 game/ 資料夾即可遊玩。

------------------------------------------------------------------------
一、準備遊戲原檔
------------------------------------------------------------------------
Waxworks 是 1992 年 Horrorsoft / Accolade 的作品。請用你合法擁有的
版本（原版磁片、或合法數位重售版）。把**所有**遊戲檔複製進本資料夾的
game/ 子目錄，至少要包含這些（大小寫不拘）：

    GAMEPC          STRIPPED.TXT    TBLLIST         XTBLLIST
    START           STATELST        ROOMSLST
    TEXT01 … TEXTxx                 (對白文字)
    TABLES01 … TABLESxx             (腳本)
    ROOMS01 … ROOMS0x               (房間)
    XTABLE01 … XTABLE0x
    *.VGA           (畫面圖像，數量很多)
    *.MUS  MUSIC.DRV  SAMPLE.DRV    WAX.FXB   ICON.DAT   MENUS.DAT

    ↑ 直接把原版整個資料夾的檔案全丟進 game/ 最省事。

game/ 裡已經放好的三個繁中檔請保留，不要刪：
    waxworks_zh16.dcjk   waxworks_zh24.dcjk   waxworks_zh.tab

------------------------------------------------------------------------
二、（選配）原版 MT-32 真實配樂
------------------------------------------------------------------------
想聽 1992 原汁原味的 Roland MT-32 配樂，把這兩個 ROM 也放進 game/：

    MT32_CONTROL.ROM     MT32_PCM.ROM

沒有 ROM 也能玩，音樂會自動改用一般 MIDI / AdLib 音色。
（ROM 屬 Roland 版權，本套件不附，請自行合法取得。）

------------------------------------------------------------------------
三、開始遊玩
------------------------------------------------------------------------
● Windows：
    雙擊 play-waxworks.bat

● macOS：
    雙擊 play-waxworks.command
    首次被 Gatekeeper 擋時：對著它按右鍵 →「打開」→ 再按一次「打開」；
    或到「系統設定 → 隱私權與安全性」按「仍要打開」。

● Linux：
    在終端機執行  ./play-waxworks.sh
    （或用檔案管理員設為可執行後雙擊）

啟動器會自動偵測 game/ 裡的遊戲、載入繁中、開啟 MT-32 配樂。
存檔放在 saves/。

------------------------------------------------------------------------
四、遊玩中的中文與友善功能
------------------------------------------------------------------------
- 全中文對白、物品、動詞列；連 WAXWORKS 標題畫面下方都有「蠟像館之謎」。
- 防拷驗證已自動跳過，不必查手冊。
- 現代玩家友善化熱鍵（隨開隨關）：
    TAB  動態地圖（走過的地方亮起，黃點寶物、紅點怪物）
    F5   立即回復生命與心靈力
    F6   把當前房間可拿取物品一鍵放進背包
    F7   無敵模式（生命/心靈力永遠滿，噴火槍等消耗品無限）
    F8   除霧・整張可達地圖一次攤開
  這些都是額外選項，想體驗原版硬核就別按。

------------------------------------------------------------------------
五、疑難排解
------------------------------------------------------------------------
- 起動後說找不到遊戲：確認遊戲原檔真的在 game/ 裡（GAMEPC 等）。
- 畫面是英文不是中文：確認 game/ 裡有 waxworks_zh16.dcjk / _zh24.dcjk /
  _zh.tab 三個繁中檔。
- 沒有音樂或音色怪：想要 MT-32 就放兩個 ROM 進 game/；沒放會用內建音色。
- macOS 打不開：見上方 Gatekeeper 說明（右鍵→打開）。
